# 素材来源与替换说明

当前版本使用一个网络场景背景，并配合网页内原创符文、药剂瓶和过关动画。

## 已接入素材

- Great Hall of Hogwarts in Hogwarts Legacy
- 来源：https://commons.wikimedia.org/wiki/File:Great_Hall_of_Hogwarts_in_Hogwarts_Legacy.jpg
- 协议：CC BY 3.0
- 用途：网页整体场景背景

## 替换规则

如果你后续找到更喜欢的哈利波特电影或《霍格沃茨之遗》素材，可以替换 `src/game-data.js` 里的 `VISUAL_ASSETS.greatHall.imageUrl`。

建议优先选择：

- 明确允许使用或有清楚来源说明的图片
- 宽屏背景图，推荐 1280px 以上
- 色调偏霍格沃茨大厅、图书馆、地牢、药剂课、公共休息室

不建议直接把来源不明的电影截图上传到 GitHub Pages 仓库，因为 GitHub Pages 是公开站点。
